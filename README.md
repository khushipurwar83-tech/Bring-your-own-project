# 🚀 AI Pro Planner

A smart command-line based productivity planner built with Python that helps you manage tasks, build habits, generate daily schedules, and track productivity using data visualization.

---

## ✨ Features

### 📋 Task Management

* Add tasks with:

  * Name
  * Duration
  * Priority (High / Medium / Low)
  * Category
  * Deadline (optional)
* Mark tasks as completed
* Automatic task rescheduling

---

### 🧠 Smart Scheduler

* Generates a structured daily schedule
* Considers:

  * Available hours
  * Task priority
  * Deadlines
  * Mood (Energetic / Neutral / Tired)
* Adds smart break intervals ☕

---

### 🔥 Habit Tracker

* Add daily habits
* Track streaks
* Build consistency over time

---

### 📊 Productivity Analytics

* Productivity score (%)
* Task completion graph (Pie chart)
* Category distribution (Bar graph)
* Productivity trend over time (Line graph)

---

### 💡 Smart Suggestions

* Alerts for:

  * High-priority delays ⚠️
  * Too many tasks (burnout warning)
* Motivational nudges to build momentum

---

## 🛠️ Tech Stack

* Python 3
* JSON (for data storage)
* Matplotlib (for graphs)

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/ai-pro-planner.git
cd ai-pro-planner
```

### 2. Install dependencies

```bash
pip install matplotlib
```

### 3. Run the project

```bash
python planner.py
```
